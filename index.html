Frequently Asked Questions

It is a long established fact that a reader will be distracted by the readable content of a page when looking at its layout.

Flex-Box is designed for one dimension layout -either a row or a column. And Grid is designed for the two dimension layout -row and column at the same time

Bootstrap is a component based CSS framework. And Tailwind is a utility based CSS framework. In Bootstrap we can get many freemade components like carousel,modal,navbar, accoidian,card etc. On the othr side in Tailwind we wont get direct components like Bootstrap but we can get a lot of utility classes.

The CSS box model refers to how HTML elements are modeled in browser engines and how the dimensions of those HTML elements are derived from CSS properties. It is a fundamental concept for the composition of HTML webpages.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages. Some HTML5 Semantic tags are 'Header, Nav, Aside ,Article, Footer'
